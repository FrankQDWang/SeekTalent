## 岗位职责

1. Context Engineering。
2. 基于 ReAct / Reflexion 等模式设计具备自主处理复杂逻辑能力的 Agent，解决长链路业务问题。
3. 通过合理的 Context Engineering 提高 Agent 行为的可控性，降低 token 成本。
4. 低延迟拟人化 Voice Agent。
5. 通过后训练实现高度拟人化、低延迟的语音对话 Agent，高效解决业务问题。

## 任职要求

1. 具备硬核工程能力。
2. 计算机相关专业，Python 功底扎实。
3. 拥抱 AI 编程，具备通过 AI 辅助实现 10 倍研发效率的能力。
4. 有较好的分析问题、解决问题能力，能将复杂的业务需求转化为清晰的模型逻辑。
5. 具备模型训练基础，熟悉主流 LLM 后训练（Post-training）的逻辑和流程。

### 加分项

- 熟悉文本-音频输入多模态模型的训练。
- 在国内外顶级 AI 会议上发表过 NLP / LLM / Agent 相关论文。
