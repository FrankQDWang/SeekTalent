## 岗位职责

- 大模型全流程开发与优化：负责大模型（语言模型、多模态模型、CV 大模型等）的后预训练、指令微调、对齐训练等；根据项目需求选择合适的算法和技术，搭建和优化模型结构和模型应用架构；处理大规模数据，进行数据清洗、标注和预处理，确保数据质量，用于模型训练和验证。
- 构建自动化 Prompt 工程框架及评估体系，能够熟练进行对话撰写及赋能培训；设计构建语义桥接架构，实现多源异构数据的高效知识迁移；开发数据质量评估模型，建立动态数据增强机制，以满足业务需求目标。
- 智能体开发与开源技术融合：基于主流大模型开发平台，如 Dify、LangChain 等设计智能体系统，集成 RAG、Agent 等核心技术，推动业务场景快速落地。
- 适配昇腾等国产生态，负责模型在昇腾平台的算子适配与性能调优，开发训练框架的混合精度策略及分布式通信优化方案，基于昇腾构建从训练到推理的全栈体系。
- 设计面向业务的高效网络结构，如 YOLO-Family、DETR、ConvNeXt、Hybrid Transformer 等，完成剪枝、量化、蒸馏、NAS 优化。

## 任职要求

- 计算机、人工智能、数学、自动化等相关专业硕士及以上学历。
- 精通深度学习框架，如 PaddlePaddle、PyTorch 等，具备 2 年以上目标检测 / 分割项目经验。
- 数学基础扎实，包括线性代数、概率论、统计学等，能够深入理解和推导算法原理。
- 熟练掌握数据结构和算法，能够高效处理和分析数据。
- 熟练使用 Python 开发，具有良好的编程习惯。
- 熟练掌握大模型智能体开发框架，了解分布式训练和量化推理技术，能够使用开源工具完成相关工作。
- 主导过检测、分类、分割、多模态等项目。
- 精通 Python / C++，编码规范，能独立编写 CUDA / NEON 算子并落地 TensorRT / ncnn / MNN 推理框架。
- 掌握 PyTorch、TensorFlow，熟悉 CNN、Transformer、DETR、YOLO 系列及轻量化设计。
- 具备团队合作精神，有一定抗压能力、责任心和独立分析能力。
- 有参与大型人工智能项目经验者优先。
- 在自然语言处理、计算机视觉等领域有实际项目成果者优先。
- 具备大模型微调和智能体开发经验，能够独立解决项目中遇到的技术难题。
- 具有从 0 到 1 搭建和训练大模型经验者优先。
- 具有多模态或智能体项目经验者优先。
