## #58 Runtime candidate identity reverse indexes

问题：`_rebuild_identity_state()` 和 `RuntimeCandidateIdentityIndex.upsert_candidate()` 存在 O(n²) 扫描。

目标：

- 为 resume_id、evidence_id、strong identity keys、候选 fuzzy bucket 建 reverse index。
- `_identity_match_score()` 只对候选 identity 集合执行。
- 保持 canonical identity、alias、conflict 行为不变。
- 覆盖大样本性能/行为测试。

验证：

```bash
uv run pytest tests/test_runtime_candidate_identity.py tests/test_runtime_state_flow.py tests/test_runtime_multi_source_round_dispatch.py
```
