## 原始 JD

职位描述

职位概述
AI Agent开发工程师负责设计、构建和优化基于大语言模型（LLM）的智能体系统，将前沿AI技术转化为实际业务价值。聚焦于智能体的感知、决策、记忆和行动等核心能力实现，推动AI Agent在自动化流程、人机交互、数据分析等场景的落地。

岗位职责
1. 系统设计与开发：参与基于大语言模型的智能体系统设计、开发与优化，构建高效、稳定的AI Agent架构。
2. 框架应用与优化：基于LangChain、LangGraph、AutoGen、Agno等主流框架，实现任务规划、工具调用、记忆管理、多智能体协作等功能。
3. 核心模块实现：负责开发工作流引擎、插件机制、Prompt管理、上下文调度等核心模块，优化Agent的响应机制与资源调度效率。
4. 技术集成：封装多模型能力接口，支持动态调用与故障转移策略；与数据平台系统等对接，实现多任务自主执行。
5. RAG系统构建：设计并实现检索增强生成（RAG）系统，覆盖知识库构建、文档切分、Embedding优化、向量检索与Rerank，负责多轮对话记忆管理优化。
6. 业务场景落地：将Agent接入真实业务系统，构建自动化工作流，优化Agent执行稳定性，监控调用成本与性能。
7. 性能保障：搭建监控、日志和告警系统，确保AI Agent后端服务的可靠性和高性能；负责系统的容器化部署与生产环境维护。

任职要求

基本要求
1. 学历背景：本科及以上学历，计算机科学、人工智能、软件工程等相关专业。
2. 工作经验：3年以上相关领域经验，具备AI Agent项目落地经验。
3. 编程能力：精通Python，熟悉FastAPI/Flask/Django等Web框架和异步编程；或掌握Go/Java/Node.js等主流后端语言。

技术能力
1. 大模型与AI智能体技术：深入理解Transformer等大模型架构，熟悉OpenAI、Claude、通义千问等主流模型的API调用和Prompt Engineering技术。
2. Agent开发框架：熟练掌握至少一种主流智能体框架，如LangChain、LlamaIndex、AutoGen、CrewAI、Agno等，并有实际的RAG或在线Agent项目经验。
3. 数据与系统集成：具备API集成、数据库操作（SQL/NoSQL）经验，了解向量数据库（如Chroma、Pinecone、Milvus）的原理与使用，能够实现检索增强生成（RAG）系统。
4. 工程化与部署：熟悉Docker、Kubernetes等容器化技术，能够进行微服务部署和高可用性保障，具备使用FastAPI或Flask暴露和部署智能体服务的能力。

软技能
1. 问题解决能力：具备优秀的问题解决和调试能力，能够独立完成从需求分析到设计、编码、测试的全流程开发。
2. 沟通协作：拥有良好的沟通能力和团队协作精神，能够与产品、算法及业务团队紧密合作。
3. 学习能力：对AI技术充满热情，具备快速学习能力，能积极跟踪行业最新动态。

加分项
1. 有LangChain、Spring AI等框架的实战经验。
2. 有个人LLM应用作品、开源项目或GitHub示例。
3. 了解机器学习或自动化工作流工具。
4. 有AI成本优化经验。
5. 有低代码/Workflow编排经验。
6. 有AI系统工程化部署经验（Docker/K8s）。
7. 有AI应用开源项目贡献或技术论文发表经历（尤其在智能体记忆方向）。
